# Cloud Log Monitor ☁️

## Description
A Python-based cloud log monitoring simulator that analyzes application logs and generates a summary report, similar to cloud monitoring services like AWS CloudWatch.

## Features
- Log analysis
- Error and warning detection
- Summary report generation

## Technologies Used
- Python
- File handling

## How to Run
python monitor.py

## Cloud Concept
Simulates cloud-based log monitoring systems.
